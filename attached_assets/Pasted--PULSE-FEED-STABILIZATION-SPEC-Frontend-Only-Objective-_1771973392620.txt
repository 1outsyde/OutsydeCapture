2️⃣ VIDEO BEHAVIOR (CRITICAL)
Video Library

Use expo-av Video component.

Playback Rules

Only ONE video can play at a time.

The currently visible index = activeIndex.

When activeIndex changes:

Pause previous video

Play new video

Autoplay MUST be enabled.

Videos MUST be muted by default.

Looping must be enabled.

Tapping video toggles pause/play.

Required Props for Video
<Video
  ref={videoRef}
  source={{ uri: post.mediaUrl }}
  shouldPlay={index === activeIndex}
  isLooping={true}
  isMuted={true}
  resizeMode="cover"
/>
Active Index Detection

Use:

onViewableItemsChanged
viewabilityConfig = {
  itemVisiblePercentThreshold: 80
}

When item becomes 80% visible → setActiveIndex(index)
